Getting Started

Adding Products for an Event

  1. First, create an Event (this is the date for the product availability)
  2. Go to Add Product to add products to the event
  3. Each product needs a Product Name, quantity, and optionally a photo

Creating a Blog Post

  1. Go to Blog Posts and click "Add"
  2. Add a title and check "Is published" when ready to make it visible
  3. Add content blocks for text, photos, or YouTube videos
  4. For YouTube videos, use just the video ID (e.g., rmojioqOlzk) not the full URL
